## Jre-8u221
>基于alpine 3.10版本,且字符集为zh_CN.UTF-8构建jre镜像
## 使用说明
> 将jre1.8安装包下载放到当前目录，运行build.sh。就可以完成构建。

## Jre1.8下载地址
